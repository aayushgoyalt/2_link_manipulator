# 2-Link Manipulator

A 2-link robotic manipulator visualization tool built with Electron and Vue 3.

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Controls

- **θ₁, θ₂**: Joint angles
- **L₁, L₂**: Link lengths
- **Start/Stop**: Animate the manipulator
- **Reset**: Restore defaults
